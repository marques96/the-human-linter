## 📋 Sobre o Projeto

Este repositório foi criado como parte da atividade prática **"The Human Linter"**, que simula um fluxo real de revisão de código em equipe utilizando Git e GitHub.

O objetivo foi praticar o papel de **Developer** e **Reviewer** em um ambiente controlado, identificando e corrigindo erros propositais em funções Python, sem executar o código — apenas com análise visual e conhecimento das boas práticas.

---

## 🎯 Objetivos da Atividade

- ✅ Simular o fluxo de trabalho com **Pull Requests**
- ✅ Identificar **erros de lógica** e **violações de estilo (PEP8)**
- ✅ Praticar **revisão de código** com feedback claro e objetivo
- ✅ Utilizar **branches separadas** para cada funcionalidade
- ✅ Documentar problemas encontrados e soluções aplicadas
